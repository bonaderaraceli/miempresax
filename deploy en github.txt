1. Crear un proyecto en una carpeta individual.
2. Iniciamos una repo con git local.
3. Iniciamos un repo en git Hub.
4. Hacemos un push del proyecto.
5. Vamos a settings en git Hub y seleccionamos en Git Hub pages.
6. Seleccionamos la rama que queremos deployar.
7. Guardamos los cambios y esperamos 5/10 minutos para poder verlo en un link.
